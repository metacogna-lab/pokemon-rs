# Getting Started

## Prerequisites

- **Rust** (latest stable) — for CLI and controller.
- **Bun** — for TypeScript agents.
- **Node 20+** (optional) — for Mintlify CLI (`mint dev`).

## Build and test

From the repo root:

```bash
# Rust workspace
cargo build --workspace
cargo test --workspace
cargo run -p cli
```

From `agents/`:

```bash
bun install
bun run lint
bun test
```

## Regenerate API client

After changing `openapi.yaml`:

```bash
openapi-generator-cli generate \
  -i openapi.yaml \
  -g typescript-fetch \
  -o ./agents/ts-client \
  --additional-properties=supportsES6=true
```

## Preview this docs site

```bash
cd docs && npx mint dev
```

Served at `http://localhost:3000` by default.
